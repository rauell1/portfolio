# Roy Okola — Portfolio

Personal portfolio and writing focused on renewable energy engineering, electric mobility infrastructure, and sustainable systems.

**Live (Vercel):** https://roy-otieno.vercel.app/

## Highlights

- Projects and case studies (energy + mobility)
- Writing / blog
- Contact + CV/resume downloads

## Tech stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- React Router
- Supabase (optional) for dynamic content

## Run locally

`ash
npm install
npm run dev
`

Open http://localhost:5173

## Build

`ash
npm run build
npm run preview
`

## Deployment

Vercel settings (recommended):
- Build command: 
pm run build
- Output directory: dist

If using Supabase, set VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY in your environment.
